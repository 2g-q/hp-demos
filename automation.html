自動化・データ処理 作品集 AIで速く正確に、品質は全件人の目で検品。

本業でAI・自動化開発(Claude Code等を活用)。日々の手作業を、ワンクリックで終わる仕組みに置き換えます。 下は実際に動くツールの出力です。数字はすべて手元で計測した実測値です。

7 種類のツールを掲載しています。いずれも「手作業なら数十分〜数時間」の作業を、実行ボタン1つに置き換えたものです。

01 SCRAPING

Webスクレイピング・情報収集の自動化

100件 → 約10秒 手作業1時間以上を短縮

公開サイトから書籍100件を約10秒で収集し、色付き見出し・フィルタ付きの整形済みExcelで納品します。転記のミスも起きません。

出力例:収集データを整形したExcel
02 GAS 自動化

GAS業務自動化(フォーム→転記→集計→通知)

毎朝30分 → 0分 サーバー代不要

フォーム回答が届いた瞬間に、整形転記・集計表の更新・担当者への内容入りメールまで全自動。毎朝の目視確認が不要になります。

左:導入前後の比較 / 右:自動化フロー
03 PDF → EXCEL

請求書・帳票PDF → Excel変換

10枚 → 0.12秒 1枚1行のExcelに

溜まったPDFを一括で読み取り、番号・日付・金額をExcelに展開。読取不能は赤マーク、金額は自動で検算し、転記事故を防ぎます。

PDF(左)を1枚1行のExcel(右)へ
04 名寄せ / CLEANSING

企業リスト名寄せ・クレンジング

59行 → 29社 重複30件を0.1秒で除去

複数ソースのCSVを表記ゆれ統一・重複除去・統合。3ソース59行を0.1秒で名寄せし、実体29社に整理します。

表記ゆれ・重複を統一(前後比較)
05 API 差分

API差分取得・変更検知

33件 → 変更16行 突合0.01秒

外部システムから商品データをAPIで取得し、前回値と突合。変更行だけを色分け表示します(緑=新規 / 黄=価格 / 青=在庫 / 赤=廃番)。

変更行のみ色分けで抽出
06 ダッシュボード

売上ダッシュボード自動生成

売上CSV → 経営ダッシュボード 期間切替で全グラフ連動

売上CSVを読み込むだけで、カテゴリ・月次・店舗別のグラフを自動生成。期間を切り替えると全グラフが連動します。

CSVから自動生成 / カテゴリ・月次・店舗別
07 テキスト分類

テキスト分類・アンケート集計

80件 → 0.031秒 80%を自動分類

自由記述をカテゴリ自動分類し、ネガポジも判定。確信度の低いものはAIで補完し、最終は全件人の目で検品します。

カテゴリ別の自動分類結果

速さだけでなく、正確さのための仕組み

1
AIで速く、人で正確に生成AIで処理を高速化しつつ、納品物は全件人の目で検品します。
2
事故を防ぐ設計読取不能は赤マーク、金額は自動検算など、間違いに気づける仕組みを入れます。
3
数字はすべて実測掲載の処理時間・件数は誇張なく、手元で計測した実測値のみです。
